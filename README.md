# AHP Business Hub

Fullstack platform gồm:
- `Frontend/`: Next.js + TypeScript (public site + admin dashboard)
- `Backend/`: NestJS + TypeScript (REST API, auth, business modules)

## Quick Start
1. Cài dependencies cho từng app.
2. Cấu hình biến môi trường từ file `.env.example`.
3. Chạy backend trước, sau đó chạy frontend.

## Tech Stack
Next.js, NestJS, TypeScript, JWT, Google OAuth, MySQL, Docker.
